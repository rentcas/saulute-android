# Saulutės treneris – Android MVP

Android programėlė, kuri per galinę telefono kamerą:

- patikrina, ar visas žmogaus kūnas telpa kadre;
- paprašo atsitraukti, atsistoti centre arba pasisukti šonu;
- po 3 sekundžių pradeda 6,5 sekundės bandymo analizę;
- įvertina kojų tiesumą, kojų išskyrimą, klubų aukštį ir nusileidimo kontrolę;
- pateikia vieną svarbiausią kito bandymo patarimą.

## Paleidimas

Atidarykite projekto aplanką su Android Studio, palaukite kol Gradle atsisiųs priklausomybes ir paleiskite fiziniame Android telefone (min. Android 8). Kamera emuliatoriuje nėra tinkama realiam bandymui.

## Svarbūs apribojimai

Tai taisyklėmis paremtas MVP, o ne sertifikuota gimnastikos vertinimo sistema. Balų ribas būtina sukalibruoti su trenerių pažymėtais vaizdo įrašais. Vaizdas šiame MVP niekur nesiunčiamas ir nėra saugomas.
